# 🚀 CryptoSpectra

## 📌 Overview

CryptoSpectra is a simple web application that helps users explore and track real-time cryptocurrency data. The app fetches live data from a public API and displays it in an easy-to-understand and visually clean format.

The main goal of this project is to practice working with APIs, JavaScript, and building a responsive user interface while creating something useful and interactive.

---

## 🎯 Purpose of the Project

This project is part of a graded assignment to demonstrate:

* API integration using JavaScript (fetch)
* Working with dynamic data
* Using array methods like map, filter, and sort
* Creating a responsive and user-friendly UI

---

## 🌐 API Used

This project uses the CoinGecko API to fetch cryptocurrency data.

**Endpoint used:**
https://api.coingecko.com/api/v3/coins/markets?vs_currency=usd

---

## 📊 Data Displayed

The application will display:

* Name of the cryptocurrency
* Current price
* Market capitalization
* 24-hour price change
* Coin logo/image

---

## ⚙️ Features (Planned)

### 🔍 Search

Users will be able to search for cryptocurrencies by name.

### 🎛️ Filter

Users can filter coins based on criteria such as price range or other properties.

### 📊 Sort

Users will be able to sort data based on:

* Price (ascending/descending)
* Market cap
* 24-hour change

### ⭐ Additional Features

* Mark coins as favorites
* Option to view more details

### 🎨 UI/UX

* Clean and modern design
* Responsive layout (works on mobile, tablet, desktop)
* Optional dark/light mode
