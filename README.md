# 🚀 CryptoSpectra | Premium Crypto Dashboard

CryptoSpectra is a high-performance cryptocurrency dashboard built using Vanilla HTML, Tailwind CSS, and JavaScript. It provides real-time market data, global statistics, and a side-by-side comparison tool for 100+ cryptocurrencies.

---

## 🌐 API Used

This project uses the CoinGecko API to fetch live cryptocurrency data.

**Endpoint:**  
https://api.coingecko.com/api/v3/coins/markets?vs_currency=usd

---

## ✨ Features

### 📊 Real-Time Market Data
- Live cryptocurrency prices and stats  
- Data fetched dynamically from API  

### 🌍 Global Stats Banner
- Total Market Cap  
- 24h Trading Volume  
- BTC & ETH Dominance  

### 📈 Dynamic Coin Table
- 🔍 Search: Filter coins by name or symbol  
- 🔃 Sorting: Sort by Rank, Price, or 24h Change  
- 📄 Pagination: Browse top 100 cryptocurrencies  
- 📉 7-Day Sparklines: SVG-based mini charts  

### ⚖️ Asset Comparison Tool
- Compare any two cryptocurrencies  
- Side-by-side analysis of metrics  

### 🎨 Premium UI/UX
- Glassmorphism effects  
- Dark theme with neon accents  
- Smooth animations and toast notifications  
- Fully responsive design  

---

## 🛠 Tech Stack

- HTML5  
- Tailwind CSS (CDN)  
- Vanilla JavaScript (ES6+)  
- CoinGecko API  
- Lucide Icons  
- Google Fonts (Outfit)

## 🚀 How to Run the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/Harsh200814/CryptoSpectra.git
   ```

2. Navigate into the folder:
   ```bash
   cd CryptoSpectra
   ```

3. Open `index.html` in your browser
