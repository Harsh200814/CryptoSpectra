# CryptoSpectra | Premium Crypto Dashboard

CryptoSpectra is a high-performance, premium cryptocurrency dashboard built entirely with **Vanilla HTML, CSS (Tailwind), and JavaScript**. It provides real-time market data, global statistics, and a side-by-side comparison tool for over 100+ cryptocurrencies.

## 🌐 API Used

This project uses the CoinGecko API to fetch cryptocurrency data.

**Endpoint used:**
https://api.coingecko.com/api/v3/coins/markets?vs_currency=usd

## 🚀 Features

- **Real-time Market Data**: Fetched directly from the CoinGecko API.
- **Global Stats Banner**: Live updates on Market Cap, volume, and BTC/ETH dominance.
- **Dynamic Coin Table**:
  - **Search**: Instant filtering by name or symbol.
  - **Sorting**: Sort by Rank, Price, or 24h Change.
  - **Pagination**: Browse through the top 100 market assets.
- **7-Day Sparklines**: Custom SVG-based price trend visualizations.
- **Asset Comparison**: Select any two coins from the dashboard to compare their metrics side-by-side.
- **Premium UI/UX**:
  - Glassmorphism effects with backdrop blur.
  - Responsive design for mobile and desktop.
  - Smooth micro-animations and toast notifications.
  - Dark-themed aesthetic with neon accents.

## 🛠 Tech Stack

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Modern styling via CDN.
- **Vanilla JavaScript**: Core logic and DOM manipulation (ES6+).
- **CoinGecko API**: Market data source.
- **Lucide Icons**: Premium iconography.
- **Google Fonts**: Outfit for a modern feel.
